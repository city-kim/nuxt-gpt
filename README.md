# Nuxt.js로 openai api 연동하여 slack GPT채팅 만들기

### .envsample 파일을 참고하여 openai api key와 슬랙 Bot User OAuth Token을 넣으십쇼

## 설치하십쇼

```bash
# yarn
yarn install

# npm
npm install

# pnpm
pnpm install
```

## 서버를 실행하십쇼

http://localhost:3000 가 기본값입니다

```bash
npm run dev
```

## 빌드하십쇼

```bash
npm run build
```

로컬에서 빌드 프리뷰를 보십쇼

```bash
npm run preview
```
